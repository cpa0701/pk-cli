> 基于vue和ivew的pk生态前端项目脚手架

## 安装

首先, 安装 [Yeoman](http://yeoman.io) 然后安装generator-pk-cli

```bash
npm install -g yo
npm install -g generator-pk-cli
```

然后生成你的项目:

```bash
yo pk-cli
```

目前仅提供vue+iview一种模版，后续将增加react,ng2;antd,icework，element等
